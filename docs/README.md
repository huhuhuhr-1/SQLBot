# SQLBot 文档中心

欢迎来到 SQLBot 文档中心！这里包含了 SQLBot 项目的完整文档，帮助您快速上手、深入开发和生产部署。

## 📚 文档概览

SQLBot 是一个基于大模型与 RAG 的智能问数系统，本文档中心按照不同用户角色和使用场景进行了分类组织：

| 文档类型 | 适用用户 | 主要内容 | 推荐阅读顺序 |
|----------|----------|----------|--------------|
| [安装指南](./INSTALLATION_GUIDE.md) | 系统管理员、运维人员 | 系统安装、环境配置、快速部署 | 1️⃣ |
| [用户指南](./USER_GUIDE.md) | 最终用户、数据分析师 | 功能使用、操作指南、最佳实践 | 2️⃣ |
| [开发指南](./DEVELOPMENT_GUIDE.md) | 开发者、贡献者 | 开发环境、代码规范、调试指南 | 3️⃣ |
| [生产部署指南](./PRODUCTION_DEPLOYMENT.md) | 运维工程师、架构师 | 生产部署、性能优化、监控运维 | 4️⃣ |
| [常见问题解答](./FAQ.md) | 所有用户 | 问题排查、故障排除、技术支持 | 5️⃣ |
