# 工具 3：数据库操作 (db-operations)

## 基本信息

**JAR 文件**：`.claude/skills/meta-ops/scripts/db-operations-tool.jar`

**用途**：DDL（创建/修改表）和 DML（增删改查）操作。

## 语法

```bash
java -jar .claude/skills/meta-ops/scripts/db-operations-tool.jar \
  -d <database_type> -u <user> -p <password> -j <jdbc_url> \
  --driver <driver_jar> --driverClass <driver_class> \
  -o <operation> [options] --output <file>
```

## DDL 操作

### 支持的操作

| 操作             | 说明  | 必需参数                    |
|----------------|-----|-------------------------|
| `create-table` | 创建表 | `--table`, `--column`   |
| `alter-table`  | 修改表 | `--table`, `--alter-op` |
| `drop-table`   | 删除表 | `--table`               |

### ALTER TABLE 操作类型

| `--alter-op`    | 说明   | 必需参数                             |
|-----------------|------|----------------------------------|
| `add-column`    | 添加列  | `--column`                       |
| `drop-column`   | 删除列  | `--old-name`                     |
| `modify-column` | 修改列  | `--column`                       |
| `rename-column` | 重命名列 | `--old-name`, `--new-name`       |
| `add-index`     | 添加索引 | `--index-name`, `--index-column` |
| `drop-index`    | 删除索引 | `--index-name`                   |
| `rename-table`  | 重命名表 | `--new-name`                     |

### 列定义格式

**格式**：`--column "name:TYPE[:option1:option2:...]"`

**支持的列选项**：

- `primary_key` - 主键
- `not_null` - 非空约束
- `unique` - 唯一约束
- `auto_increment` - 自增（MySQL）
- `default(VALUE)` - 默认值

### 数据类型映射

| 类型     | PostgreSQL                                       | MySQL                | Oracle                                    |
|--------|--------------------------------------------------|----------------------|-------------------------------------------|
| 整数（自增） | `SERIAL` / `BIGINT GENERATED ALWAYS AS IDENTITY` | `INT AUTO_INCREMENT` | `NUMBER GENERATED BY DEFAULT AS IDENTITY` |
| 字符串    | `VARCHAR(n)` / `TEXT`                            | `VARCHAR(n)`         | `VARCHAR2(n)`                             |
| 时间戳    | `TIMESTAMPTZ`                                    | `TIMESTAMP`          | `TIMESTAMP`                               |
| 布尔值    | `BOOLEAN`                                        | `TINYINT(1)`         | `NUMBER(1)`                               |

## DML 操作

### 支持的操作

| 操作       | 说明 | 必需参数                          |
|----------|----|-------------------------------|
| `insert` | 插入 | `--table`, `--values`         |
| `update` | 更新 | `--table`, `--set`, `--where` |
| `delete` | 删除 | `--table`, `--where`          |
| `select` | 查询 | `--table`                     |

### 输出格式

`-f text|json|table|markdown`（默认 text）

## 使用示例

### 创建表

```bash
skill_session_init meta-ops create-table "创建 users 表"

java -jar .claude/skills/meta-ops/scripts/db-operations-tool.jar \
  -d postgresql -u sqlbot -p sqlbot \
  -j jdbc:postgresql://localhost:17082/testdb \
  --driver .claude/skills/database-drivers/drivers/postgresql-42.3.8.jar \
  --driverClass org.postgresql.Driver \
  -o create-table \
  --table users \
  --column "id:SERIAL:primary_key" \
  --column "username:VARCHAR(100):not_null" \
  --column "email:VARCHAR(255):not_null:unique" \
  --column "created_at:TIMESTAMPTZ:default(now())" \
  --output "$(skill_session_report_path create_result.md)"
```

### 添加列

```bash
skill_session_init meta-ops alter-table "添加 phone 字段"

java -jar .claude/skills/meta-ops/scripts/db-operations-tool.jar \
  -d postgresql -u sqlbot -p sqlbot \
  -j jdbc:postgresql://localhost:17082/testdb \
  --driver .claude/skills/database-drivers/drivers/postgresql-42.3.8.jar \
  --driverClass org.postgresql.Driver \
  -o alter-table \
  --table users \
  --alter-op add-column \
  --column "phone:VARCHAR(20)" \
  --output "$(skill_session_report_path alter_result.md)"

skill_session_finish "表结构修改完成"
```

### 插入数据

```bash
java -jar .claude/skills/meta-ops/scripts/db-operations-tool.jar \
  -d postgresql -u sqlbot -p sqlbot \
  -j jdbc:postgresql://localhost:17082/testdb \
  --driver .claude/skills/database-drivers/drivers/postgresql-42.3.8.jar \
  --driverClass org.postgresql.Driver \
  -o insert \
  --table users \
  --values '{"username":"testuser","email":"test@example.com"}' \
  --output "$(skill_session_report_path insert_result.md)"
```

### 查询数据（表格格式）

```bash
java -jar .claude/skills/meta-ops/scripts/db-operations-tool.jar \
  -d postgresql -u sqlbot -p sqlbot \
  -j jdbc:postgresql://localhost:17082/testdb \
  --driver .claude/skills/database-drivers/drivers/postgresql-42.3.8.jar \
  --driverClass org.postgresql.Driver \
  -o select \
  --table users \
  --limit 10 \
  -f table \
  --output "$(skill_session_report_path query_result.md)"
```
